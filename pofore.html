Web design is the process of planning, designing, and implementing a website to be functional and user-friendly. It involves working on every attribute of the website that people interact with, including the look and feel, functionality, and user experience.

UI&UX Design

UX and UI design are two related professions that often work together. UX stands for "user experience" and UI stands for "user interface". UX designers focus on the interaction between a user and a product or service. UI designers focus on the aesthetics of a product's user interface.
